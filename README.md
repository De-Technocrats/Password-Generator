# Password-Generator
A python tool for generating strong password

## How to install:
Open termux and type the commands below

```

apt update

apt upgrade

apt install git

apt install pip

git clone https://github.com/De-Technocrats/Password-Generator

cd Password-Generator

apt install python3

python3 setup.py

```

## Now run the tool by typing:

```
python3 password.py

```
